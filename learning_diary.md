## 🛠️ 指令速查与逻辑拆解

| 指令 | 英文单词 | 形象含义 | 动作拆解 |
| --- | --- | --- | --- |
| `cd` | Change Directory | 切换房间 | 切换到目标文件夹 |
| `touch` | Touch | 轻轻一触 | 新建一个空白文件 |
| `git init` | Initialize | 装监控 | 在当前文件夹开启版本监控 |
| `git add` | Add | 打包包裹 | 将文件放入待处理清单 |
| `git commit` | Commit | 封箱并标签 | 将修改固化为一次历史存档 |
| `git push` | Push | 发射快递 | 将本地存档推送至云端 |
| `nano` | Nano | 简易写字台 | 打开终端自带的文本编辑器 |

## 🛠️ 指令速查表 (Cheat Sheet)

| 指令 | 形象含义 | 核心作用 |
| --- | --- | --- |
| `cd` | 切换房间 | 进入指定文件夹 |
| `mkdir` | 盖新房 | 新建一个文件夹 |
| `git init` | 装监控 | 开启代码版本监控 |
| `git add` | 打包包裹 | 将文件放入暂存区 |
| `git commit -m "..."` | 封箱存档 | 永久保存当前代码快照 |
| `git push` | 快递发射 | 将本地存档上传至 GitHub |
| `git remote -v` | 地图检查 | 查看云端连接地址是否正确 |
